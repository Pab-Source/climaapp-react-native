## Comenzando 🚀

_Ejecuta_

`yarn install`
**Yarn**

### Pre-requisitos 📋

```
node 14.4.0
yarn 1.22.4
expo 3.21.5
```

### Instalación 🔧

**Ejecutar AVD emulador**
`expo start ó yarn start`

## Ejecutando las pruebas ⚙️

_Para ejecutar las pruebas_

`yarn test`
